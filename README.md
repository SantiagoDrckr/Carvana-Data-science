# Proyecto de Análisis de Datos: Caso Carvana

El caso Carvana es un proyecto de análisis de datos sobre el mercado de venta de vehículos usados en Estados Unidos. El objetivo es predecir si es una mala compra en función de sus características, como el año, la marca, el modelo, el kilometraje, el estado y la ubicación.

El proyecto utiliza un conjunto de datos proporcionado por la empresa Carvana, que se dedica a la compra y venta de coches usados en línea. Puedes encontrar el conjunto de datos en [Kaggle](https://www.kaggle.com/datasets/salilchitnis/carvana).

Se aplican técnicas de limpieza, exploración, visualización y modelado de datos para obtener insights y generar un modelo predictivo. El proyecto está desarrollado en Python.
